Informe de Pruebas de Rendimiento – API Docintelia

Fecha: 27 de julio de 2025

Tipo de prueba realizada

Se llevó a cabo una prueba de carga y estrés sobre la API del sistema Docintelia, para evaluar su rendimiento bajo condiciones intensas de uso concurrente.

Objetivo

Medir la capacidad de respuesta, estabilidad y robustez de la API al recibir múltiples solicitudes simultáneas.

Herramienta utilizada

Se utilizó Locust, una herramienta de prueba de carga basada en Python, que permite simular usuarios concurrentes e interpretar estadísticas en tiempo real desde una interfaz web.

Conclusión

La API de Docintelia demostró una capacidad destacada para manejar alta concurrencia sin errores. El sistema respondió con gran rapidez: el 99% de las solicitudes se procesó en menos de 51 milisegundos. No se registraron fallos, y el comportamiento fue estable durante toda la prueba.

Estos resultados validan que la API está preparada para ambientes exigentes, asegurando eficiencia y fiabilidad en situaciones de uso intensivo.
